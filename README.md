# Dashboard de Analise de Dados com IA

Dashboard que analisa dados de vendas automaticamente e responde perguntas usando Inteligencia Artificial.

## Funcionalidades

- Le qualquer ficheiro CSV automaticamente
- Gera 4 graficos de analise de vendas
- IA responde perguntas sobre os dados em portugues
- Analise de vendas por mes, produto e lucro

## Tecnologias

- Python 3
- Pandas
- Matplotlib
- Google Gemini AI
- Requests

## Como usar

1. Instala as dependencias: pip3 install pandas matplotlib requests
2. Configura a API Key: export GEMINI_API_KEY="a_tua_chave"
3. Coloca o teu ficheiro CSV na pasta com o nome vendas.csv
4. Executa: python3 dashboard.py
5. Faz perguntas sobre os dados

## Exemplo de pergunta

Qual foi o mes com mais vendas?
